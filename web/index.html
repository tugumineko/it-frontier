贴一段代码，看模型按上下文区分同一个符号的不同含义（如 key 是字典键还是密钥）。

Agent 判读（ecnu-max）

用上方代码 + 这个符号，让 ecnu-max 判断它在此处的含义、依据哪些行、置信多少。

多义符号案例

选一个符号，星海里标出它两个义项的落点和各自最近邻。
